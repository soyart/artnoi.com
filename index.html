About this webs(h)ite

This is where I write my blog and host some public cheat sheets.

I’m not a front-end web dev, so no, this static site does not take 5 seconds to load, nor does it eat half of your RAM.

The website is divided into 4 locations: /blog blog, /cheat geek cheat sheet, /noob noob cheat sheet.

FOSS <3

I’m a huge advocate of free and open-source software.

The entire website is built and deployed using webtools, a repository of simple shell scripts and GitHub Actions workflows to build and deploy HTML websites from Markdown files.

All pages on artnoi.com are simple HTML documents with one single separate style cheat with no external resources, and no cookies are collected. No JavaScript code is used on this website.

Credits

The website partially uses monospaced TTF Hack, which is my desktop and terminal font face, with blue-ish color scheme from Iceberg.

artnoi.com is served using OpenBSD httpd(8) and relayd(8).

As of Nov 2023, artnoi.com is served using GitHub Pages, and built using webtools. webtools internally uses ssg.
